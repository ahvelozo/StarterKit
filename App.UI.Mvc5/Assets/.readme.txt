
This folder contains the client side scripts, styles, images and miscelaneous content required to properly render the application.

Client side libraries are managed by Visual Studio Library Mananger (LibMan) instead of nuget packages.

For more details, check the main github repository: https://github.com/aspnet/LibraryManager

The following are script templates that can be found under the '.info' folder:

- script.template.js / Used in page specific scripts
- service.template.js / Used in application specific scripts
